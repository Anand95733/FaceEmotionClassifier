About This Project

This project demonstrates a simple Face Emotion Classifier built using a Convolutional Neural Network (CNN) in TensorFlow/Keras, exposed via a FastAPI backend, and consumed by a Flask web frontend.

Key Features:

  • Emotion Classification: Identifies "Happy" or "Sad" emotions from uploaded face images.
  • MERN-like Stack: Utilizes MongoDB (conceptual, if storing data), Express (Node.js/FastAPI equivalent for backend), React (Flask/HTML/JS equivalent for frontend), and Node.js (FastAPI/Python backend).
  • RESTful API: The backend provides a clean API for image prediction.
  • Real-time Feedback: Displays predicted emotion and confidence instantly.
  • User-Friendly Interface: A simple and intuitive web interface for image uploads.

Technologies Used:

  • Frontend: HTML, CSS, JavaScript, Flask
  • Backend: Python (FastAPI), TensorFlow, Keras, NumPy, Pillow
  • Deployment: Uvicorn (ASGI server for FastAPI)
